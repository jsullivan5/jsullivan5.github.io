Doing Things Media

Built in React.js on top of AWS cloud services, this site serves a video upload page for a social media personality with millions of followers. The site uses Amazon's DynamoDB, API Gateway, Lambda and S3 to store user data and content. This architecture provides a cost effective and scalable solution to growth in the user base and a safeguard against downtime during viral moments.

Quizzam

This group project, built with a React, Router, Redux stack with an Express and Knex.js back end, is a real time quizzing application that allows students and teachers to connect via web sockets to get a pulse check on the room. This app provides an open source, software solution to classroom polling.

TextSmart UI

This project removes the ambiguity of text messaging by implimenting Watson and Twilio on a React/React Router/ Redux stack. Send a text message to our number and receive sentiment analysis on it's content. Users can review their history and view submitted content on the site.

TextSmart API

This RESTFUL API coordinates with the UI and intagrates the IBM Watson and Twilio APIs. Built with Node and Express, it uses MongoDB and Express Session for data and session persistance.

Connect The Dots

This project was built to play with maps. Built with Leaflet.js on top of React and Redux, users can map latitude and longitude to a leaflet map. Rendered points are clickable and can connect the dots oon the map. Data is persited with MongoDB.

Foto Finder

This resful API is designed for use in photo applications, where enthusiasts can provide EXIF details about their photos, and connect photos to locations and cameras. It uses JWTs for authentication and returns all responses in JSON format.
